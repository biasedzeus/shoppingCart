# Shoping Cart using FakerJS and Pexels
## Features
 - Add Items to the cart
 - Items can only be added once
 - CheckOut from Cart
 - On successfull checkout recieve a toast.
 
 ## Technologies Used:
 
  - ReactJs
  - Reactstrap
  - Bootstrap
  - Axios
  - PexelsAPI
  - FakerAPI
